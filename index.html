Clientes e Servidores

Para entender um pouco melhor o mundo do HTML e desenvolvimento web, precisamos entender quem são os clientes e os servidores. Os clientes são responsáveis pelo início da comunicação com o servidor. O cliente mais conhecido é o navegador. Quando abrimos um site no navegador, uma requisição é disparada pela internet para o servidor. O servidor é responsável por receber essa requisição e devolver com uma resposta para o cliente. O servidor pode ser um ou mais computadores com um único endereço. Geralmente, o servidor não envia dados se o cliente não realizar a requisição primeiro. (Fonte: DEVMEDIA1)

O que é HTML?

HTML é sigla para HyperText Markup Language, que significa em português: Linguagem de Marcação de Hipertexto trecho. O HTML é o componente base para websites, pois por meio dele é possível a criação de sites com imagens e vídeos. (Fonte: TOTVS, 20202)

Portanto, HTML não é uma linguagem de programação e sim um arquivo de marcações.

Principais Tags de HTML

Tags são marcações que orientam a estrutura do site, como o tamanho, a fonte, as quebras de linhas e inserção de links, vídeos e imagens.

As tags mais conhecidas são:

  • head: Define a parte que não é mostrada para o usuário;
  • body: Define a parte que é mostrada para o usuário;
  • title: Marca o título da pagina web;
  • h1 e h2: Marca títulos e subtítulos do conteúdo da página web
  • strong: Marca um conteúdo para ser em negrito;
  • p: Marca parágrafos;
  • i: Marca um conteúdo para ser em itálico;
  • u: Marca um conteúdo para ser sublinhado;
  • a: Marca inserção de link;

Fonte: TOTVS, 20202; W3SCHOOLS3.
